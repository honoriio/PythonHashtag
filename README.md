# PythonHashtag

Repositório de estudos em Python criado durante meu processo de aprendizado com o curso da Hashtag.

## Sobre este repositório

Este repositório reúne exercícios, testes práticos e pequenas implementações desenvolvidas enquanto eu estudava os fundamentos da linguagem Python.

Aqui estão registrados conteúdos como:

- variáveis e operações
- estruturas condicionais
- estruturas de repetição
- listas
- tuplas
- dicionários
- funções
- métodos de string

O objetivo deste repositório não é apresentar um projeto final pronto, mas sim documentar minha evolução na linguagem por meio de exercícios e prática constante.

## Objetivo

Usei este espaço para:

- praticar lógica de programação
- reforçar a sintaxe do Python
- entender estruturas de dados básicas
- ganhar familiaridade com entrada, processamento e saída de dados
- criar disciplina de estudo com código

## Estrutura

Os arquivos e pastas estão organizados por tema de estudo, acompanhando os assuntos praticados ao longo do curso.

Exemplos de conteúdos presentes no repositório:

## Observações
